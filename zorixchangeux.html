交易頁面使用體驗研究

工作項目

UX使用體驗研究｜問卷設計｜統計分析

專案角色

擔任產品設計| 獨立製作

專案介紹

以問卷調查、SUS易用性測試、滿意度分析、統計分析及簡易訪談等方式，進行交易頁面操作的使用體驗研究，深入考察使用者在線上交易操作中所遇到的問題以及重視的項目。依據問卷結果、SUS測試資料及訪談洞察結果，進行交易頁面使用體驗及UI設計優化，並統整使用者對於產品的主要期待，定義產品發展目標，以此作為產品團隊發展考量重點。

研究設計

  • 受測者隨機分組，進行 組間實驗
  • 分別測試電腦版 UI ModelA / ModelB 及 手機版UI ModelA / ModelB
  • 為避免學習效應，每位受測者僅安排一種Model做測試，ex.電腦版 UI ModelA + 手機版 UI ModelA
  • 受測者有相同操作流程任務：選擇賣出幣別、選擇買入幣別、輸入金額、確認交易、完成交易
  • 紀錄指標：任務操作時間（秒）、事後滿意度（SUS）、事後訪談問卷

實驗結果

  • 電腦版 UI ModelA：操作時間顯著較短、滿意度顯著較高
  • 手機版 UI ModelB：操作時間顯著較長、滿意度顯著較低
  • 最終電腦版 環境優先採用 UI A、手機版 環境優先採用 UI B
  • 事後訪談問卷：發現受測者對於介面使用順暢度及即時度會影響受測者對於交易所的信任程度，且對於即時性及便利性會有較高的期待。
